앱 설치 방법

별도 앱 설치 없이 크롬 브라우저에서 홈 화면에 추가하면 앱처럼 사용할 수 있습니다.

🤖 안드로이드 (크롬)
1크롬(Chrome) 브라우저로 앱 주소에 접속합니다.
2주소창 아래 "앱 설치" 배너가 나타나면 탭합니다.
배너가 없으면: 우측 상단 메뉴(⋮) → 홈 화면에 추가 또는 앱 설치
3설치를 탭하면 홈 화면에 아이콘이 생깁니다.
4이후에는 홈 화면의 아이콘을 탭해서 실행합니다.
🍎 아이폰 (사파리)
1사파리(Safari) 브라우저로 앱 주소에 접속합니다. (크롬 불가)
2하단 가운데 공유 버튼(□↑)을 탭합니다.
3메뉴에서 "홈 화면에 추가"를 탭합니다.
4이름 확인 후 추가를 탭합니다.
💡설치 후에는 일반 앱처럼 전체 화면으로 실행되며, 푸시 알림도 받을 수 있습니다.
⚠아이폰에서는 반드시 사파리로 접속해야 홈 화면에 추가할 수 있습니다. 크롬으로는 설치가 되지 않습니다.

① 출발 보고
1현장으로 출발할 때 🚚 출발 버튼을 탭합니다.
2도착 예상 시간을 확인하고 필요하면 수정합니다. (기본값: 30분 후 반올림)
3출발 확인 버튼을 탭하면 문자가 자동으로 발송되고 상태가 이동중으로 바뀝니다.
📋출발 보고 복사 버튼을 누르면 카카오톡에 붙여넣기 할 수 있는 출발 보고 양식이 복사됩니다.
💬문자 발송 후 재발송 버튼으로 ETA를 수정하여 재발송할 수 있습니다.
⚠출발을 잘못 눌렀다면 출발 취소 버튼으로 되돌릴 수 있습니다.
② 작업 시작 보고
1현장 도착 후 작업을 시작할 때 도착 · 작업시작 버튼을 탭합니다.
2(선택) 예상 물량과 예상 작업시간을 입력합니다.
3(선택) 현장 사진을 찍어 첨부합니다.
4작업시작 버튼을 탭하면 작업 시작 시간이 기록되고 상태가 진행중으로 바뀝니다.
③ 작업 완료 보고
1작업이 끝나면 완료 사진을 찍어 첨부합니다. (카메라 버튼 또는 갤러리에서 선택)
2특이사항이 있으면 입력합니다.
3최종 물량을 확인하고 입력합니다.
4작업완료 버튼을 탭합니다. 상태가 완료로 바뀌고 종료 시간이 기록됩니다.
📋작업 보고 복사 버튼을 누르면 아래 형식의 작업보고서가 클립보드에 복사됩니다.

처리장처리장 이름 선택HK (강서천일 / 기타 선택 가능, 기타는 직접입력)
처리비용납부한 처리비직접 입력

상성폐기물 종류혼합 (목재 / 왈가닥 / 기타 선택 가능)

기타 기능
🔒 비밀번호 변경
1상단 헤더의 🔒 비밀번호 버튼을 탭합니다.
2현재 비밀번호, 새 비밀번호 (4자 이상), 새 비밀번호 확인을 입력합니다.
3변경 버튼을 탭합니다.
⚠비밀번호를 잊어버린 경우 관리자에게 초기화를 요청하세요.

뒤로가기 버튼스마트폰 뒤로가기 버튼 → 로그아웃 확인 팝업 표시

7
자주 묻는 질문
Q일정이 안 보여요
상단 날짜 선택기가 오늘 날짜로 되어 있는지 확인하세요. 다른 날짜로 설정되어 있으면 오늘 일정이 보이지 않습니다.
Q출발 버튼을 잘못 눌렀어요
일정 상세 화면에서 출발 취소 버튼을 눌러 대기 상태로 되돌릴 수 있습니다.
Q완료 처리 후 사진을 추가하고 싶어요
완료 상태에서도 수정 버튼을 눌러 사진·특이사항·최종물량을 수정할 수 있습니다.
Q알림이 두 번 와요
브라우저를 강력 새로고침(Ctrl+Shift+R) 하여 서비스 워커를 최신 버전으로 업데이트하면 해결됩니다.
Q처리비 보고를 잘못 제출했어요
잘못된 건은 관리자에게 삭제 요청을 해주세요.
